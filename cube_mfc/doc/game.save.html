Simulation Save/Restore

The simulation can be saved to a file using the File/Save or File/Save As menu items.
The simulation can be restored from a previously saved file using  the File/Open menu item.

A new simulation can be started at any time using the File/New menu item.   Choosing this option will discard the current simulation, so save if first if you want it.

The following states are saved in simulation files:
  • Cube orientation
  • Cube section rotations
  • Zoom scale
