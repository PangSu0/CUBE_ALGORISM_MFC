Rubic's Cube Simulation

Written by Ross Wolin/Mystic Industries ©2003


This is a Win32/MFC  application which simulates the rotation and manipulation of a Rubic's cube using OpenGL graphics.   It was written mostly as a demo, but might also be a fun game for people that like Rubic's Cubes.
